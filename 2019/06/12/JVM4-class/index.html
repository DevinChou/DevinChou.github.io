4.4 类加载器

JVM的类加载系统十分复杂，这里只实现一个简易的类加载器。
ClassLoader结构体

loadBasicClass加载Class类，在LoadClass方法中会递归加载父类和接口，并给已加载的所有类生成对应的Class对象

loadPrimitiveClass加载void,boolean,byte,short,int,long,char,float,double8个基本变量类型，并为每一个生成对应的Class对象（如int.class）

4.4.1 类加载LoadClass

LoadClass方法加载类，如果类已加载，则返回go Class，如果尚未加载，则判断是数组类还是非数组类，并在加载完后生成对应类型的Class对象

loadArrayClass加载数组类

loadNonArrayClass加载非数组类，先读取类的字节，将解析字节转为go Class放入方法区,然后进行验证和初始化，包括给每个实例变量field和静态变量field分别设置slotId,初始化常量的值

readClass根据类完整限定名，读取类的字节数组

4.4.2 defineClass构建Class

defineClass根据字节数组构建go Class，随后去解析其父类和接口，放入方法区

parseClass将字节转成classfile文件，然后调用go Class的newClass方法将classfile转成 go Class
